# **Turtle Quest**

Welcome to Turtle Quest! Embark on an exciting journey as a brave little turtle navigating a challenging world. Evade obstacles, cross dangerous paths, and complete your ultimate quest!

(This was a school project made to demonstrate mastery of the SCRUM process and python turtle library)

---

## Game Features
* **Adaptive Controls**: Simple and responsive control mechanics designed for all skill levels.
* **Level Progression**: Dynamic difficulty scaling—as you clear levels, obstacles speed up and become trickier.
* **Lightweight & Fast**: Built cleanly to launch instantly with minimal dependency overhead.

## Built With
* **Language**: Python 3.14
* **Libraries/Frameworks**: turtle graphics module (Standard Library)
